# 🎨 ASL Butique Logo Ekleme Rehberi

## Adım 1: Logo Dosyasını Kaydet

1. Gönderdiğiniz logo resmini bilgisayarınıza kaydedin
2. Dosya adını `logo.png` olarak değiştirin
3. Bu dosyayı şu klasöre kopyalayın:
   ```
   web/public/logo.png
   ```

## Adım 2: Kod Otomatik Güncellenecek

Logo dosyasını kaydettikten sonra, kod otomatik olarak logoyu kullanacak şekilde ayarlandı.

## Alternatif: Online URL Kullan

Eğer logoyu bir yere yüklediyseniz (örn: imgur, imgbb), URL'sini kullanabiliriz:

```jsx
<img 
  src="LOGO_URL_BURAYA" 
  alt="ASL Butique" 
  style={{ height: 50 }}
/>
```

## Logo Boyutları

- **Önerilen yükseklik**: 50-60px
- **Arka plan**: Siyah (logoda zaten var)
- **Format**: PNG (şeffaf arka plan için)

## Sonraki Adım

Logo dosyasını `web/public/logo.png` olarak kaydettikten sonra bana haber verin, kodu güncelleyeceğim!
