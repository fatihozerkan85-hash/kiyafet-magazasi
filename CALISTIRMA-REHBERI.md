## 🎯 Nasıl Çalıştırılır?

### 1. Backend'i Başlat
```bash
cd backend
node server.js
```
Backend http://localhost:5000 adresinde çalışacak

### 2. Frontend'i Başlat
```bash
cd web
npm start
```
Frontend http://localhost:3000 adresinde açılacak
