# 🚀 Deploy Komutları

Git bir onay sorusu soruyor. Lütfen terminal'de şu komutları sırayla çalıştırın:

## Adım 1: Dosyaları Ekle
```bash
git add .
```
**Soru gelirse:** `A` (Yes to All) tuşuna basın

## Adım 2: Commit
```bash
git commit -m "Boyner tarzı kategori menüsü - sade tasarım, emoji ikonları kaldırıldı"
```

## Adım 3: Push
```bash
git push origin main
```

## ✅ Tamamlandı!

Vercel otomatik olarak deploy edecek (1-2 dakika).

Site: https://kiyafet-magazasi.vercel.app

## 🔍 Yapılan Değişiklikler

- ✅ Kategori menüsünden emoji ikonları kaldırıldı
- ✅ Sade, minimal tasarım (Boyner tarzı)
- ✅ Transparent arka plan (seçili olmayan kategoriler)
- ✅ Hover efekti eklendi
- ✅ Border-radius azaltıldı (4px)
