🚀 Seed Data - Otomatik Çalıştırma

⚠️ DİKKAT: Bu sayfa otomatik olarak seed data'yı çalıştıracak!

Yöntem 1: Otomatik (Önerilen)

Aşağıdaki butona tıklayın, sayfa otomatik olarak seed data'yı çalıştıracak:

Yöntem 2: Manuel (Browser Console)

Adım 1: F12 tuşuna basın (Developer Tools açılır)

Adım 2: "Console" tab'ına gidin

Adım 3: Aşağıdaki kodu kopyalayıp Console'a yapıştırın ve Enter'a basın:

fetch('https://kiyafet-magazasi-backend.vercel.app/api/admin/seed-data', { method: 'POST', headers: { 'Content-Type': 'application/json' } }) .then(r => r.json()) .then(d => { console.log('✅ BAŞARILI!', d); alert('Seed data eklendi!\nKategoriler: ' + d.counts.kategoriler + '\nKampanyalar: ' + d.counts.kampanyalar + '\nÜrünler: ' + d.counts.urunler); }) .catch(e => { console.error('❌ HATA:', e); alert('Hata: ' + e.message + '\n\nDeployment henüz tamamlanmamış olabilir. 1-2 dakika bekleyip tekrar deneyin.'); });

Yöntem 3: PowerShell

Windows PowerShell'de çalıştırın:

Invoke-WebRequest -Uri "https://kiyafet-magazasi-backend.vercel.app/api/admin/seed-data" -Method POST -ContentType "application/json" | Select-Object -ExpandProperty Content
