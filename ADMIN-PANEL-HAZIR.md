# ✅ Tam Özellikli Admin Paneli Hazır!

## 🎯 Özellikler

### 1. 📊 Dashboard
- Toplam ürün, sipariş, kullanıcı sayısı
- Toplam gelir
- Son siparişler
- Düşük stoklu ürünler

### 2. 🎉 Kampanya Yönetimi (Mevcut)
- Kampanya ekleme/düzenleme/silme
- Aktif/Pasif yapma
- Resim, tarih, renk ayarlama

### 3. 🛍️ Ürün Yönetimi (YENİ!)
- Ürün listesi (tablo)
- Yeni ürün ekleme
- Ürün düzenleme
- Ürün silme
- 4 resim URL'si
- Fiyat, stok, kategori
- Beden ve renk seçenekleri

### 4. 📁 Kategori Yönetimi (YENİ!)
- Kategori listesi
- Yeni kategori ekleme
- Kategori düzenleme
- Kategori silme
- Emoji ekleme

### 5. 📦 Sipariş Yönetimi (YENİ!)
- Sipariş listesi
- Sipariş detayları
- Durum güncelleme
- Kargo takip numarası

### 6. 🎟️ Kupon Yönetimi (YENİ!)
- Kupon listesi
- Yeni kupon oluşturma
- İndirim tipi ve miktarı
- Aktif/Pasif

---

## 🚀 Kullanım

1. **Giriş Yap**
   ```
   https://kiyafet-magazasi.vercel.app/admin.html
   Kullanıcı: admin
   Şifre: admin123
   ```

2. **Sekme Seç**
   - Dashboard, Kampanyalar, Ürünler, Kategoriler, Siparişler, Kuponlar

3. **İşlem Yap**
   - Ekle, Düzenle, Sil butonlarını kullan
